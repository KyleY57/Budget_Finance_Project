# Budget_Finance_Project
Excel Finance Project WIP

Purpose: Strengthen Excel understandings and applications. 

Action:
Version 1:
Created a simple dashboard, that takes in monthly financial information to make a 3 year projection on savings and budgetting. Took a big emphasis on tables within excel, and extracting data from the table. Worked with cross sheet referencing as well as simple macros for accessibility. Worked through unique ways to add randomness within data, and finally used INDEX and XLOOKUP (in the similar fashion) to search table for specific values, as well as SUBTOTAL to work with values that may be filtered out. Took advantage of data validation and conditional formatting for visual ease. 


Version 2:
After gaining confidence, and a bit more understanding with tables and excel, next goal was to make the budgetting more realistic and interactive. More in detph interactive panel, allowing for more accurate simulations. Applied BC's progressive tax system using real values for 2026. Automatically calculated salary values, tax rates, and other financial metrics for analysis. A few small macros for ease of use and quality of life purposes, but minimal. More emphasis on new savings rate and bank balance, with charts that can visualize net worth growth. Focused on a more realistic simulation of expenses throughout a 1 year period.

Future Adjustments (WIP):
Version 1 let me get familiar with excel basics, while Version 2 put a heavier emphasis on calculations within excel. 
VBA macros and manipulating dataset would be my next goal. 
Possible ideas for macros would be: Annuity calculations using the savings we collected, possible future promotion adjustments to salary and inflations to adjust values.
As far as manipulating datasets go, using a table that we generated may not be the best idea. Future project idea may be to use an existing REAL dataset then manipulate that rather than manipulate a dataset that was simulated from information.
